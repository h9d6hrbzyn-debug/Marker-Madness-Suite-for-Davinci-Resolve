Tool 01
Marker Madness
Full Marker Management For DaVinci Resolve.
Now There's Nothing You Can't Do With A Marker.
Features
BATCH CHANGE ANYTHING!!
  • Browse all markers with color, timecode, name, note, and duration
  • Inline editing — click any name or note cell to edit directly
  • Sort and filter by color, keyword, or timecode range
  • Batch rename with find/replace, prefix/suffix, case, trim, and counter
  • Right-click the color swatch to change color; right-click name or note to edit
  • Copy markers across timelines and paste at playhead
  • Float on Top keeps the window visible over Resolve
  • Auto-refreshes when you click back in
Pro Tips
Step Counter — the middle number trick
Counter starts at Start × Step. Set Start:1, Step:10, Digits:3 → 010, 020, 030. Perfect for VFX shot numbering.
After Counter position
Choose "After counter" in the Add field to place text after the number: ShotName_010_VFX in one pass.
Marker Renamer auto-refreshes
Select different markers, click back into the Renamer panel — preview updates automatically to reflect your new selection.
Tool 02
Clip Renamer Pro
Batch Rename Media Pool Clips And Timelines With Live Preview.
Features
  • Rename clips, timelines, or both in one pass
  • Find & replace, add prefix/suffix, or replace entire name
  • Counter with configurable digits, start, step, and position
  • Trim characters from the start or end of names
  • UPPERCASE / lowercase / Title Case / Remove digits
  • Live before→after preview updates as you type
  • Undo last batch or restore all original names
  • Cmd+Z keyboard undo
Pro Tips
Start × Step = first counter value
Counter begins at Start × Step, not Start. Start:1 Step:10 → 010, 020, 030. Start:1 Step:1 → 01, 02, 03.
"After counter" — text after the number
Add position "After counter" places your text right after the digits: ClipName_010_VFX, ClipName_020_VFX.
Nothing in the preview? Select clips first.
Only clips selected in the Media Pool appear. Select in Resolve, then hit Refresh.
Tool 03
Track Command
Rename, Recolor, And Manage Tracks In The Current Timeline.
Features
  • Audio Tracks | Video Tracks | Rename Operations — three-column layout
  • Same full rename engine as Clip Renamer Pro
  • Live rename preview column before you apply
  • Add audio tracks: Mono, Stereo, or Adaptive — with color
  • Add video tracks with color
  • Delete tracks with full undo support
  • Right-click any track to set color from a palette
  • Select All / Deselect All for bulk operations
  • Save and load track name sets as .txt files
  • Refresh re-fetches the current timeline — even if you switched in Resolve
  • Cmd+Z keyboard undo
Pro Tips
Step Counter — same rule applies
Start:1, Step:10, Digits:3 → 010, 020, 030. Great for Dia_010, Dia_020 track naming.
Switched timelines? Hit Refresh.
Track Command re-fetches the current project and timeline on Refresh — tracks update instantly even after switching.
"After counter" on tracks too
Add: _Boom, position: "After counter" → Dialogue_010_Boom, Dialogue_020_Lav in one Apply.
